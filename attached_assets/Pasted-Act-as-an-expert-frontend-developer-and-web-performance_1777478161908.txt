Rename every static asset using an 8-character MD5/SHA content 
hash appended to the filename before the extension:

  BEFORE:

    assets/video/hero-bg-mobile.mp4

    assets/video/hero-bg.[hash8].mp4
    assets/video/hero-bg.[hash8].webm
    assets/video/hero-bg-mobile.[hash8].mp4
    assets/images/hero-fallback.[hash8].jpg
    assets/css/style.[hash8].css
    assets/js/canvas.[hash8].js

Generate the hash using Node.js:

  const crypto = require('crypto');
  const fs = require('fs');

  function hashFile(filePath) {
    const buffer = fs.readFileSync(filePath);
    return crypto.createHash('md5')
      .update(buffer)
      .digest('hex')
      .slice(0, 8);
  }

Create a build script (scripts/hash-assets.js) that:

  - Outputs a manifest.json mapping old names to new names
